# deportes-temuco
Pequeño proyecto de gestión de datos.

Donde ponemos una pagina en línea de temuco y podemos organizar/gestionar datos de los partidos del club.

Ideas de futuras actualizaciones :

-Cada jugador contara con una foto de perfil-
-Se pondran las formaciones ocupadas en cada partido-
-Se habilitara una pagina de espectador donde solo pueden ver los datos sin poder editarlos-
-Se aumentara la informacion disponible del jugador, de que club viene, hasta cuando tiene contrato, altura, posiciones mas precisas y no tan genericas-


Se actualizaran los minutajes de forma automatica, poniendo el 11 incial de cada partido y poniendo las substituciones de ese partido para que se actualice de forma automatica
